LODQ: Overview

Welcome to the Web interface of the Linked (Open) Data Query porcessor (LiDaQ).
A service which allows the execution SPARQL queries over the content of Web sources. The service is provided by a collaboration between the Digitial Enterprise Research Institute (DERI) and the Karlsruher Institute for Technologie (KIT) and hosted and maintained by the Semantic Web Search Engine (SWSE) Team.

Web Service

We provide an easy to use Query form for our service. The current implementation should be thread-safe and able to handle a couple of request in parallel with the current allocated hardware. However, please bear in mind that this is a research project and that we have limited hardware resources and bandwidth - use the service with responsibility.

Alpha Status

The service is in an early alpha status and under continues development. You will expect from time to time all certain behaviors of a prototypical Web service - this includes that our service

  • is not available or responsive (most likely cause: maintenance downtime or internal server errors);
  • does not return any results, even if it should (most likely cause: code errors or problems with other Web servers);
  • does not handle and communicate errors in the best possible way (most likely cause: implementation of server side error handling);
  • is missing detailed documentation about the usage (most likely cause: guess ;)).
Please inform us using the contact provided in the footer if you experience some of the above mentioned cases or other strange behavior of the LiDaQ service .
